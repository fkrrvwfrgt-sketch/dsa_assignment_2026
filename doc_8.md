# Documentation: Task 8 - Sorting Algorithms

## (a) Data Structure Definition
Standard integer arrays are used to store and manipulate the data during sorting.

## (b) Function Descriptions
- `bubbleSort()` / `selectionSort()` / `insertionSort()`: Basic $O(n^2)$ sorting methods.
- `quickSort()`: A recursive divide-and-conquer algorithm that uses a pivot.

## (c) main() Organization
The `main()` function generates 10 random numbers using `rand()`, stores them in an array, and demonstrates the Quick Sort algorithm on that array.

## (d) Sample Output
Generated Random Numbers: 41 67 34 0 69 24 78 58 62 64 

Sorted using Quick Sort: 0 24 34 41 58 62 64 67 69 78
